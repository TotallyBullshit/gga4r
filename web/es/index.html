Gga4r, Algoritmos Genéticos para Ruby

Introducción

Gga4r es una implementación de algoritmos genéticos en Ruby fácil de usar:

  1. Coge la clase a evolucionar y defínele los métodos fitnes, recombine y mutate: 
    
	class StringPopulation<Array
	    def fitness
	      self.select { |pos| pos == 1 }.size.to_f / self.size.to_f
	    end
	
	    def recombine(c2)
	      cross_point = (rand * c2.size).to_i
	      c1_a, c1_b = self.split(cross_point)
	      c2_a, c2_b = c2.split(cross_point)
	      [StringPopulation.new(c1_a + c2_b), StringPopulation.new(c2_a + c1_b)]
	    end
	
	    def mutate
	      mutate_point = (rand * self.size).to_i
	      self[mutate_point] = 1
	    end
	end
  2. Crea un objeto del tipo GeneticAlgorithm con la población a evolucionar. 
    
    def create_population_with_fit_all_1s(s_long = 10, num = 10)
        population = []
        num.times  do
            chromosome = StringPopulation.new( Array.new(s_long).collect \
                                                { (rand > 0.2) ? 0:1 } )
            population << chromosome
        end
        population
    end

    ga = GeneticAlgorithm.new(create_population_with_fit_all_1s)
  3. Llama a la función eval tantas veces como quieras y observa como ha evolucionado. 
    
	100.times { |i|  ga.evolve }
	p ga.best_fit[0]

Instalar

  1. Ejecutar: 
    
	gem install gga4r
  2. Añadir dependencias en las cabeceras: 
    
	require "rubygems"
	require "gga4r"

Atención!

Ten en cuenta que Gga4r añade los métodos shuffle!, each_pair y split en la clase Array.

Documentación

La documentación se puede generar con la herramienta rdoc a partir del código fuente con el siguiente comando: 


  rdoc README lib
